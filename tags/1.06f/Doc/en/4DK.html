Development kit

A development kit is available on my web page. This will let you build more add-ons for BeHappy. If you want to use it, download it and unzip the archive. Launch BeHappy and ask the BeHappy Documentation add-on to do an update. It should use the development kit's documentation instead of BeHappy's.

Please note that if you download a new version of BeHappy, you should also download the new version of the development kit. (they have the same version number)
