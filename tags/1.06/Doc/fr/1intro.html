Introduction

BeHappy est un navigateur de documentation HTML pour le système d'exploitation BeOS. Il affiche une liste contenant des liens, ou un index, et une vue HTML. Quand vous cliquez sur un item de la liste, la description de l'item apparait dans la vue HTML.

Quelles sont les principales caractéristiques de BeHappy?

  • La liste et la vue HTML sont dans la même fenêtre. Pas besoin de passer d'une fenêtre à une autre!
  • BeHappy peut s'adapter lui même à une nouvelle version de la documentation. Si vous téléchargez un nouveau BeBook. Il suffira juste de faire une mise à jour, et BeHappy mettra à jour automatiquement ses liens et son index!
  • Les add-ons vous permettent de naviguer dans toutes les documentations HTML!
  • BeHappy est controlable par un autre programme. Il peut être utilisé pour afficher l'aide d'une application.

Les add-ons sont conçus avec un kit de développement que l'on peut trouver sur mon site web.
