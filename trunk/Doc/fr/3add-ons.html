Les Add-Ons

Pour utiliser d'autres documentations au format HTML, vous aurez besoin d'installer un add-on pour BeHappy. Certains sont disponibles sur mon site web.

Pour installer un nouvel add-on, il suffit juste de le mettre dans le répertoire "Add-on" de BeHappy. Alors vous trouverez le nom de la documentation dans le menu "Livre" de BeHappy. Si le nom n'apparait pas, cela peut venir des ces différentes raisons:

  • Le fichier est corrompu, et il n'est pas reconnu comme un add-on. Essayez de le télécharger à nouveau.
  • L'add-on est plus vieux ou plus récent que la version de BeHappy que vous utilisez (dans ce cas, vous verrez un message d'erreur). Essayez toujours d'avoir les dernières versions de BeHappy et des add-ons que vous utilisez. J'espère que je serais capable de maintenir une compatibilité avec toutes les versions de BeHappy, mais je ne le garantis pas, malheureusement.

Dans un premier temps l'add-on est chargé, et la documentation HTML est analysée de manière à construire l'index. Si vous installez une nouvelle version de la documentation, vous devrez effectuer une mise à jour de l'index. Pour cela, sélectionnez la documentation dans le menu "Livre", et sélectionnez "Mettre à jour" dans le menu "Action".

Vous pouvez aussi avoir plus d'information sur l'add-on avec l'item "A propos de l'add-on" dans le menu "Action".
