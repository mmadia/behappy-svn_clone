Utilisation de BeHappy

La première chose que vous devez faire est d'ouvrir un livre. Pour cela, allez dans le menu "Livre" et sélectionnez un des items. L'introduction ou le sommaire du livre apparait dans la fenêtre, et sur la gauche vous verrez les contrôles de BeHappy. Voici les contrôles du BeBook:


Le menu déroulant vous permet de choisir l'index que vous avez besoin d'utiliser. Avec le BeBook, vous avez quatre index:

  • Par kit
  • Par héritage
  • Index des fonctions
  • Index des sujets
Sélectionnez un index et la/les liste(s) en dessous change(nt). Vous pouvez avoir plusieurs types de listes:
Liste simple
Il suffit juste de sélectionner un item dans la liste.		 Liste double
Sélectionnez un item dans la liste du haut, et plus de détails apparaitront dans celle du bas. Sélectionnez un item dans la liste du bas pour afficher le texte correspondant dans la fenêtre.		 Liste avec une recherche
Tapez quelques chose dans la zone de texte du bas, et la liste va afficher tout les items qui commencent par ce que vous avez tapé.		 Option "Contenant"
Avec cette option de sélectionnée, tout les items contenant ce que vous avez tapé sont affichés dans la liste.
Parfois vous verrez un bouton "Suivant". Il recherche si l'item que vous avez sélectionné dans la liste du haut existe quelque part ailleurs. C'est très pratique dans l'index "Par Héritage" dans le BeBook, avec les classes qui utilisent l'héritage multiple.

Vous pouvez choisir la langue utilisée par BeHappy avec le menu 'Language'. Pour l'instant BeHappy peut avoir des menus en Français, Anglais et Allemand. Vous pouvez ajouter votre propre traduction si vous voulez. Pour cela, regardez le fichier BeHappy.loc dans le dossier de BeHappy. Un petit mode d'emploi vous dira comment faire. Il faudra également éditer les fichiers *.loc dans le dossier Add-ons.
