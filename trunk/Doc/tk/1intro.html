Introduction

BeHappy is a HTML documentation browser for the BeOS system. It displays a list with some links, or an index, and a HTML view. When you click on an item in the list, the item's description appear in the HTML view.

What are BeHappy's main features?

  • The list and the HTML view are in the same window. No need to keep on swapping between two different windows!
  • BeHappy can adapt itself to a new version of the documentation. If you download a new BeBook, just click on update, and BeHappy's links and index are automaticaly updated!
  • Add-on support can let you browse any HTML documentation!
  • BeHappy is controllable by another program. It can be used to display help about an application.

Add-ons are made with a development kit that can be found on my page.
